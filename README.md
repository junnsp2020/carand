# カランド

URL: http://18.180.81.161/


## 概要
フリマサイト「カランド」では、商品の売買だけでなく、交換も可能となっています。

## バージョン
ruby 2.5.7
Rails 5.2.4.1

## 機能一覧  
・出品機能 [取扱説明 : A  をご参照ください]  
・検索機能 [    〃  : B  をご参照ください]  
・購入機能 [    〃  : C-1をご参照ください]  
・交換機能 [    〃  : C-2をご参照ください]  
・通知機能 [    〃  : D  をご参照ください]  
・通報機能 [    〃  : E  をご参照ください]  


## 取扱説明:

## A. 出品 → 売れた場合の流れ    

**(1) 「マイページ」 → 「商品を出品する or カメラマーク をクリック」**
<img width="800px" src="https://user-images.githubusercontent.com/58461408/77818420-76d2cd80-7115-11ea-8c78-98515d3309fc.png">
<img width="800px" src="https://user-images.githubusercontent.com/58461408/78642693-19493880-78ee-11ea-87fe-bafcdee84082.png">
      
**(2) 出品商品の詳細を記述 → 「出品する」をクリック**  
交換の可否の項目で「交換可能」にしておけば、他のユーザーから  
交換リクエストがくる可能性があります。
<img width="800px" src="https://user-images.githubusercontent.com/58461408/78635985-59a2b980-78e2-11ea-8580-0e3ada377c98.png">
      
**(3) 出品処理は以上です**  
自分が出品した商品には「～さんが出品した商品です」と表示され
購入はできません。
<img width="800px" src="https://user-images.githubusercontent.com/58461408/77818465-e779ea00-7115-11ea-93db-c12f926965aa.png">  
**(4)商品が売れた場合**  
「商品が購入されました」とメッセージが表示されます。購入者の入金完了まで待機してください。
<img width="800px" src="https://user-images.githubusercontent.com/58461408/77819424-20b65800-711e-11ea-9afb-bc47613bbe1e.png">  
**(5)購入者が入金を終えた場合**  
出品者のヘッダーにある「売れた！」に数字が表示されます。  
この数字は出品者がやるべきことの数字を表しています。クリックして出荷報告をしてください。  
(ヘッダーの詳細については「D章.ヘッダー(通知)」を参照してください)
<img width="800px" src="https://user-images.githubusercontent.com/58461408/77822003-1736eb00-7132-11ea-8f58-044a05dc1603.jpg">
  
**(6)購入者が受取評価を終えた場合**  
購入者が受取評価を終えると、出品者はお相手の評価ができます。  
これにて取引は完了です。  
<img width="800px" src="https://user-images.githubusercontent.com/58461408/77819479-77239680-711e-11ea-93d6-54532f6c739b.png">
## B. 商品を検索する    

**(1)「ヘッダーの検索窓」 or 「カテゴリーからもお探しできます」 をクリック**  
(検索窓にキーワードを入れないで虫眼鏡マークを押すと、全ての商品を検索できます)
<img width="800px" src="https://user-images.githubusercontent.com/58461408/78636606-a63ac480-78e3-11ea-9f99-86a4f5eea652.png">

**(2)自分が見たい商品をクリックしてください**  
商品一覧には、「販売中」「交換可」「売切」のタグが設定してあります。  
「カテゴリーからもお探しできます」から検索した場合は、そのカテゴリー内の  
交換で人気を集めている商品が分かります
<img width="800px" src="https://user-images.githubusercontent.com/58461408/78640942-7bed0500-78eb-11ea-935a-e000b7e02f50.png">

## C. 購入 or 交換 → 取引の流れ    

### C-1.「購入する場合]    
**(1)「購入ページへ進む」をクリックし取引確定ページへ進んでください**  
<img width="800px" src="https://user-images.githubusercontent.com/58461408/77819386-ca491980-711d-11ea-85b2-4a8622cd8dc8.png">

**(2)「購入を確定する」をクリック**  
<img width="800px" src="https://user-images.githubusercontent.com/58461408/77820480-f9638900-7125-11ea-901e-0816af461435.jpg">  
**(3)入金報告をしてください**  
入金報告完了後、出品者は発送の手続きに入ります。  

<img width="800px" src="https://user-images.githubusercontent.com/58461408/77819410-03818980-711e-11ea-91d9-9bfcc30915e7.png"> 

**(4)出品者より出荷報告があった場合**  
ヘッダーの「買った！」に数字が表示されます。  
この数字は購入者がやるべきことの数字を表しています。クリックして受取評価をしてください。  
(ヘッダーの詳細については「D章.ヘッダー(通知)」を参照してください)  
これにて取引は完了です。  
<img width="800px" src="https://user-images.githubusercontent.com/58461408/77830559-aa8d1200-716c-11ea-8326-134140aafbc4.jpg">    

### C-2.「交換する場合」    
**(1)「この商品は交換可能です」をクリック**  
<img width="800px" src="https://user-images.githubusercontent.com/58461408/77820897-151c5e80-7129-11ea-9235-5d7324ac39b0.png">    
**(2)「交換リクエストページに提案商品を記述」**  
お相手の商品をGETするために、あなたが提案できる商品を記入し、投稿してください。  
<img width="800px" src="https://user-images.githubusercontent.com/58461408/77820987-cfac6100-7129-11ea-9e6c-84c6137203be.png">    
**(3)「WANTED!」画面の確認**  
下の画面が表示されると交換リクエストの投稿は完了です。  
交換が承認されることを期待して待ちましょう!  
<img width="800px" src="https://user-images.githubusercontent.com/58461408/77821050-5feaa600-712a-11ea-827d-b29e915ce591.png">    
**(4)リクエストが許可された場合**  
ヘッダーに「リクエストが許可されました」と表示されますのでクリックしてください。  
クリックするとオレンジ色のボタンが出ていますので、  
押すと、交換取引の確定ページへと進めます。  
<img width="800px" src="https://user-images.githubusercontent.com/58461408/78640539-e5204880-78ea-11ea-9389-b6269a8ce025.png">
<img width="800px" src="https://user-images.githubusercontent.com/58461408/77821113-0171f780-712b-11ea-9eab-e501fe43e406.png">    
**(5)「交換取引を確定する」をクリックし、取引を確定させる**  
<img width="800px" src="https://user-images.githubusercontent.com/58461408/78641368-1c432980-78ec-11ea-8d52-908c443f7788.png">
**(6)送り状番号を送り合う**  
メッセージ機能を使用し、送り状番号を互いに通知してください。  
確認が終わったら、「確認完了」のボタンを押します。  
<img width="800px" src="https://user-images.githubusercontent.com/58461408/77821268-2adf5300-712c-11ea-95df-8674cfbd3885.png">  
**(7)商品到着後、お相手の評価をしてください**  
お相手への評価をもって取引は完了となります。      

### D. ヘッダー(通知)      

#### ※ヘッダーには２つの通知機能があります※

**～バッジの通知機能～**  
取引上やるべきことがある時に数字で教えてくれます。  
数字が表示された場合は早めのご対応をお願いします。  
(※ 下図、赤矢印をご参照ください)      

**～メッセージの通知機能～**  
必ずしもやるべきことではないが、確認が望ましい場合は  
ヘッダーにメッセージとして表示されます。  
(ex. 交換リクエストをもらった場合、取引相手からのメッセージが届いた場合、  
商品の購入はされたが、入金待ちの場合等に表示されます)  
もし表示された場合は、ヘッダーをクリックして  
該当のアクションへと進んでください。(※ 下図、青矢印をご参照ください)      
<img width="800px" src="https://user-images.githubusercontent.com/58461408/77824680-d053f000-7147-11ea-8d29-6156cb780592.png">

### E. 通報      

万が一、不適切な商品を発見された場合は【 通報ボタン 】よりお知らせください。  
頂いた情報を元に厳正な対応を致します。  
カランドはお客様に安心してご利用頂けるフリマサイトを目指しています。  
<img width="800px" src="https://user-images.githubusercontent.com/58461408/77828684-42383380-7160-11ea-89cc-b526f67d0fab.png">

### F. マイページメニュー      

マイページメニューでは、  
・「購入した商品」  
・「出品中の商品」、「売れた商品」  
・「自分が送った交換リクエスト」、「他のユーザーからもらった交換リクエスト」  
・「ほしいものリストに登録した商品」  
を閲覧できます。      
<img width="800px" src="https://user-images.githubusercontent.com/58461408/78640044-03d20f80-78ea-11ea-86b6-16e856a7f2a1.png">
      
**交換リクエスト【もらった】をクリックしたページです。魅力的な提案があった場合はリクエストを許可しましょう**    
<img width="800px" src="https://user-images.githubusercontent.com/58461408/77827399-b40c7f00-7158-11ea-9573-5db4d54dbe30.png">   
また【 個人情報詳細 】をクリックすれば  
登録情報の確認、及び編集ができます。      

【 カランド内 残高 】には売れた商品の利益総額が表示されています。  
振込申請をクリックし、受け取りたい金額を  
入力してください。  
<img width="800px" src="https://user-images.githubusercontent.com/58461408/78639310-ac7f6f80-78e8-11ea-869e-70103034f8e0.png">
